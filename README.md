# Описание
Данный проект предназначен, для использования его в качестве BackEnd-а.

Для каждого пользователя, в зависимости от тарифа (по умолчанию базовый) создается база, 
данная база, в базе можно создать `N` таблиц и по определенным маршрутом отправлять `post`, `get`,
 `delete`, `put`, ... запросы и получать ответ. 

Все действия происходят на сервере API.

`Для кого это?` - прежде всего, для тех, кто не разбирается в `BackEnd`, а просто хочет сразу иметь базу данных и добавлять,
создавать, редактировать данные в таблице без использования `SQL` запросов.
